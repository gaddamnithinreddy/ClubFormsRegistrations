# Club Forms & Registrations

A modern form builder and registration system for clubs and organizations, built with React, TypeScript, and Supabase.

## Features

- 🎨 Modern and responsive UI with dark mode support
- 📝 Rich text form builder with multiple field types
- 🔒 Secure authentication and role-based access
- 📱 Mobile-friendly design
- 📊 Form response management
- 🔗 Easy form sharing with QR codes
- 📅 Calendar integration for events

## Tech Stack

- React
- TypeScript
- Tailwind CSS
- Supabase
- Vite
- Framer Motion

1. Clone the repository
2. Install dependencies: `npm install`
3. Set up environment variables
4. Run development server: `npm run dev`
